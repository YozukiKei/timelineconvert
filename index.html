使い方

        
TSV → markers.tsv で保存 → Premiere のマーカーパネル ☰ →「マーカーを読み込み」

        
EDL → markers.edl で保存 → ファイル → 読み込み

        
XML → markers.xml で保存 → Premiere で File / ファイル → Import / 読み込み

        
※ EDL は旧来のASCII系フォーマットです。この版では日本語ラベルを安全に扱えるよう、EDL内はASCIIセーフ化しつつ、日本語は \\uXXXX 形式でコメントに保持します。

        
※ XML は Final Cut Pro 7 互換の xmeml 形式で、シーケンスマーカーとして書き出します。

        
※ ブラウザで直接開くだけで動きます。インストール不要。
